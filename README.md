# CircleCI Demo Application

This is a simple Node.js application created for CircleCI demonstration purposes.

## Features

- Express.js web server
- Health check endpoint
- Jest test suite
- CircleCI pipeline configuration

## Getting Started

```bash
npm install
npm start
```

## Testing

```bash
npm test
```

## CircleCI

This project is configured to run on CircleCI. The pipeline will:
1. Install dependencies
2. Run tests
3. Build the application

View the pipeline at: https://app.circleci.com
